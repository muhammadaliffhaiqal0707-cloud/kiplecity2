# KipleCity Enhanced Website

Open `index.html` in a browser. Keep the `assets` folder beside the HTML, CSS, and JS files.

Enhancements included:
- Apple-style smooth reveal on scroll
- Cinematic hero with parallax
- Glassmorphism navigation
- Magnetic buttons
- Hover tilt cards
- Active section navigation
- Responsive mobile menu
- Deck image gallery with lightbox
